# CafeMap
주변 카페를 조회하고 방문 예약 신청이 가능한 서비스

## 사용 기술 및 개발 환경

- Java 17
- Spring Boot 2.7.3
- Gradle
- MyBatis
- MySQL
- GitHub Actions
- JaCoCo
